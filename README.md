## 🚀 Key Projects

### 💰 [EasyTaxation – Tax & Financial Services Platform](https://easytaxation.in)
> **Tech:** Next.js · JavaScript · Tailwind CSS

A multi-page financial services website for a tax consultancy serving gig economy workers (Ola, Uber, Swiggy, Zomato, Blinkit) and MSMEs across India. Built with SSR for SEO optimization, WhatsApp API integration, and a Book Free Consultation booking flow.

---

### 🏨 [Bison River Resorts – Luxury Resort Booking Website](https://bisonriverresorts.com)
> **Tech:** Next.js · JavaScript · Tailwind CSS

Premium hospitality website for a luxury riverside resort in Dandeli, Karnataka. Features multi-page layout with hero sliders, gallery, package listings, lazy-loaded images, local SEO, and direct booking CTAs.

---

### 📱 Loan Application System
> **Tech:** React Native · Java Spring Boot · REST APIs · Google Maps API

Comprehensive mobile loan application system with multi-stage approval workflow, role-based admin panel (RBAC), real-time loan status tracking, Google Maps integration for location tracking, and a notification system for automated reminders.
